**SPRT** (Simplified PxGrid, RADIUS and TACACS toolkit) is a toolkit for RADIUS/TACACS+ flows test. It allows to generate big amount of RADIUS and TACACS+ sessions in a row (several thousands).

Supported protocols/flows:

- MAB
- PAP
- PEAPv1 with EAP-MSCHAPv2
- EAP-TLS
- CoA with different results
- Guest flows with ISE:
  - Hotspot
  - Login with provided credentials
  - Self-registration
- TACACS+

Supports integration with ISE over pxGrid.

Full docs are available [here](https://github.com/cisco-open/sprt/wiki)
